## Bamazon mySQL database and App

## Instructions
 - run node bamazonCustomer.js in gitBash and select the id of the item you would like to purchase.
 - Select the number of items you would like to purchase and view your total.

## Requirements
- Use mySQL database
- Must be able to create, replace, update, and destroy information.
- execute node application which references database items.

## Technologies Used
- mySQL
- node.js
- npm modules
	- mysql
	- inquirer
- Javascript callback functions
